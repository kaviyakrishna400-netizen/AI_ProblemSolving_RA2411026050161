# AI Problem Solving Assignment

## Problem 5: Map Coloring
Algorithm:
- Constraint Satisfaction Problem (CSP)

Description:
Assign colors to regions so no adjacent regions have same color.

Sample Output:
A -> Red
B -> Green
C -> Blue
D -> Red


## Problem 8: Smart Navigation System
Algorithms:
- Breadth First Search (BFS)
- Depth First Search (DFS)

Description:
Finds path between start and goal nodes.

Sample Output:
BFS Path: A-B-D-F
DFS Path: A-C-E-F
